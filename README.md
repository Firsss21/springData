## Spring Data

Простое приложение по применению стандартного функционала Spring Data

-----

### **Описание:**

- manyToMany, manyToOne, oneToMany, oneToOne

- @Transactional, @Query

- Eazer / Lazy loading
-----

### **Технологии:**

  - База данных MySQL

  - Spring Data, Boot, MVC

  - Maven, Lombok

-----
